## Implementation Notes for Agents

**Design direction (for design-agent):**
- Use the default Neutral Modern design system
- Hero: gradient from accent (#2F6FEB) to accent-80%
- Cards: white surface, 1px border, 12px radius
- Typography: Inter, 600 headings, 400 body
- Animations: subtle fade-in on scroll, no autoplay video

**Build notes (for saas-builder):**
- Mobile-first responsive grid
- Lazy-load testimonial images
- Form capture on primary CTA with email + goal selection
- A/B test ready: swap hero CTA text variants

**SEO notes (for seo-agent):**
- Target keywords: "AI learning platform", "ATS resume builder", "portfolio generator", "job-ready skills"
- Meta description: "Learn skills, build projects, create ATS resumes, and generate portfolios — all with AI. Join 15,000+ students getting job-ready."
- Schema: Product + EducationalOrganization
