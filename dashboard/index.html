Quick Start

Up and running in 3 steps

1

Clone the repo

git clone https://github.com/Vishwajeetsrk/Agent-Team-Skills.git
or gh repo clone Vishwajeetsrk/Agent-Team-Skills

2

Set up memory

macOS/Linux: bash memory-sync/sync-memory.sh init
Windows: .\memory-sync\sync-memory.ps1 init

3

Start your AI session

Paste the generated prompt into Claude Code, claude.ai, Antigravity, or Open Code.

Pro tip: Run sync-memory.ps1 pull at the start and push at the end of every session.
