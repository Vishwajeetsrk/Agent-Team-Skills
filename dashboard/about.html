About This Project

19 specialized AI agents that work as one team — from validation to deployment, with persistent memory.

The Problem

Every AI session starts from scratch. Mistakes repeat across projects. There's no validation step — builders invest weeks in features nobody wants. Each AI tool needs different setup, different prompts, different workflow. Quality varies wildly between sessions.

The Solution

Agent Team Skills bundles 19 specialized agent skills into one orchestrated system. A single CONNECT-PROMPT activates the entire team. The CEO agent validates ideas before anything is built. The memory system logs mistakes once and prevents them everywhere. The Golden Flow ensures consistent quality every session.

The Golden Flow

Every project follows a strict order: validate → research → plan → build → design → test → legal → SEO → deploy → log. Three agents are never skipped: test-agent (security/QA), security-agent (for Web3 projects), and memory-agent (learning).

Built for Any AI

The system works with Claude Code, claude.ai, Antigravity, Open Code, and any AI that supports skill or instruction files. No proprietary platform lock-in. The memory system uses git — your data stays in your control.

MIT Licensed — Free to Use

Clone the repo, set up memory, paste the prompt. Your AI team is ready in 5 minutes. No subscriptions, no API keys required (beyond your chosen AI platform).

Open Design

30 brand-grade design systems ship with the repository — from Stripe and Vercel to Apple and OpenAI. Each includes complete tokens, CSS variables, and component specs. The design-agent references these before any UI work, ensuring brand consistency across every project.

The Creator

Built by Vishwajeet to solve the fundamental problem of AI workflows: starting from zero every session. This system encodes months of learnings about what works — and what doesn't — into reusable skill definitions that any AI can load.
